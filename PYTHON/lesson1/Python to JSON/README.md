# JSON to Python & Back  
Repository: [HexSecGuru/IEKAKMH](https://github.com/HexSecGuru/IEKAKMH.git)

A simple Python example demonstrating how to convert between Python dictionaries and JSON strings using Python’s built-in `json` module.

---

## Description  
This script shows how to:  
- Create a Python dictionary (`dict`).  
- Convert it into a JSON-formatted string using `json.dumps()`.  
- (Optionally) Convert JSON back to a Python dictionary using `json.loads()`.  
- Print the types and values before and after conversion.

It’s designed as a beginner-friendly project, perfect for understanding serialization and working with JSON data in Python.

---

## What You’ll Learn  
- Using Python’s `json` module.  
- Converting **Python → JSON** using `json.dumps()`.  
- Converting **JSON → Python** using `json.loads()`.  
- Understanding the difference between a native Python object (e.g., dict) and a JSON string.
